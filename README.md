# Anonymous Network

> The internet remembers what the world forgets.

## Overview

Anonymous Network is an experimental privacy-focused network accessed through the AN Client.

The project is designed around anonymity, privacy, decentralized services, archives, investigations, digital artifacts, and distributed network infrastructure.

Anonymous Network aims to provide a space where users can explore information, archives, and specialized services while maintaining strong privacy protections and minimal data collection.

---

## Core Principles

### Privacy First

Anonymous Network is built with privacy as a fundamental requirement.

### Minimal Data Collection

The network should collect as little user information as possible.

### Distributed Infrastructure

Services may be hosted across multiple nodes rather than relying on a single central server.

### Open Development

The project is developed openly and continuously improved by contributors.

### Information Preservation

Anonymous Network contains archives, records, investigations, digital artifacts, and historical data that may otherwise be difficult to discover.

## Vision

Anonymous Network is not intended to replace the Internet.

Its goal is to create an independent privacy-focused ecosystem where archives, investigations, digital artifacts, and specialized services can exist within a unified network accessible through the AN Client.
